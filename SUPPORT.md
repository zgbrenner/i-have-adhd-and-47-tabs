# Support

Choose the channel that matches the problem so it reaches the right place.

## Installation or usage question

Use [GitHub Discussions](https://github.com/zgbrenner/i-have-adhd-and-47-tabs/discussions) for:

- help installing the skill;
- questions about Claude, ChatGPT, Codex, or Copilot behavior;
- examples of how you use the skill;
- translation coordination;
- early ideas that are not ready for an issue.

## Reproducible project problem

Use the [issue forms](https://github.com/zgbrenner/i-have-adhd-and-47-tabs/issues/new/choose) for:

- a broken download or ZIP layout;
- a repeatable behavior problem;
- outdated platform instructions;
- validation or packaging failures.

Search existing issues before opening a new one.

## Security concern

Use [private vulnerability reporting](https://github.com/zgbrenner/i-have-adhd-and-47-tabs/security/advisories/new). Do not publish credentials, exploit details, or malicious package contents in a public issue.

## Platform account or billing problem

Contact the relevant platform's official support. This project cannot access or change Claude, ChatGPT, OpenAI, GitHub, or Copilot accounts, subscriptions, workspace permissions, or feature rollouts.

## Medical questions

This project is a response-formatting tool, not medical advice, diagnosis, or treatment. Questions about ADHD assessment or care should go to a qualified healthcare professional.
